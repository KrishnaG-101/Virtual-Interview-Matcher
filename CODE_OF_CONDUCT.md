# Code of Conduct

## Standards

- Be respectful and constructive in all interactions
- Critique code, not people
- Accept feedback graciously
- Prioritize what is best for the project over individual preferences

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Publishing others' private information without consent
- Deliberate sabotage or intentional introduction of security vulnerabilities
- Disruptive behavior in discussions or code reviews

## Enforcement

Project maintainers reserve the right to remove or edit any contribution, and to take action against any participant whose behavior violates these standards. Actions may include warning, temporary or permanent ban from the project, or escalation to appropriate authorities.

## Scope

This code applies to all project spaces: the repository, issue tracker, discussions, and any project-related communication channels.
